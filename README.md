CLC Games

Plataforma web desarrollada como proyecto final del ciclo
formativo de Desarrollo de Aplicaciones Web (DAW).

Tecnologías utilizadas:
- PHP
- MySQL
- JavaScript
- HTML5
- CSS3

Funcionalidades principales:
- Registro y autenticación de usuarios
- Catálogo de juegos con filtros
- Sistema de compra de juegos
- Biblioteca personal de juegos
- Panel de administración
- Integración de PayPal en modo sandbox

Instalación:

1. Importar la base de datos desde el archivo estructura.sql
2. Configurar la conexión a la base de datos
3. Ejecutar el proyecto en un entorno Apache (XAMPP o LAMP)
